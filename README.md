#  Todo List App

A simple and interactive Todo List web application built using HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently by adding, marking, and deleting tasks.

---

##  Features

-  Add new tasks
-  Mark tasks as completed
-  Delete individual tasks
-  Clear all completed tasks (optional feature)
-  (Optional) Local storage support to save tasks
-  Responsive design for mobile and desktop

---

##  Technologies Used

- HTML5 – Structure of the app
- CSS3 – Styling and layout
- JavaScript (ES6) – Functionality and DOM manipulation

---

## ⚙️ How It Works

1. User enters a task in the input box.
2. Clicking the **Add button** adds the task to the list.
3. Tasks can be:
   - Marked as completed ✔️
   - Deleted ❌
4. (Optional) Tasks are stored in local storage so they remain after refresh.
